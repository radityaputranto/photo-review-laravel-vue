Design a professional photography business web application called "FotoApp" 
with a clean, modern aesthetic. 

Visual Style:
- Light theme with soft gradient backgrounds (white to slate-50 or gray-50, 
  subtle blue-purple gradient accents)
- Typography: Inter or Plus Jakarta Sans font family
- Color palette: white base, slate-700 for text, blue-600 as primary accent, 
  rose-400 as secondary accent for photo selections
- UI components styled with Tailwind CSS and Flowbite component library
- Cards with subtle shadow (shadow-sm), rounded-xl corners
- Avoid generic AI-looking layouts — use intentional whitespace, 
  asymmetric grids where appropriate, and refined micro-details
- Sidebar navigation for admin, bottom or top navbar for customer
- No heavy borders, prefer spacing and shadow for separation

The app has two roles: Admin (photographer) and Customer.
